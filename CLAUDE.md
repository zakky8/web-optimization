# Web Optimization Knowledge Base

## What This Repo Is
A structured reference for building top 1% 3D, animation, and performance-optimized websites.
All data sourced from primary case studies, Awwwards winners, and production benchmarks (2024-2026).

## Folder Structure
- `01-3d-animation/` - Rendering pipeline, textures, shaders, scroll, canvas setup
- `02-mobile/` - iOS/Android limits, thermal throttling, context loss, touch events
- `03-performance-metrics/` - Real benchmark numbers and Core Web Vitals targets
- `04-resources/` - Case studies, reading list, primary sources
- `05-checklists/` - Ship-ready checklists for desktop and mobile

## How to Use
Each file is self-contained. Start with `05-checklists/` for a quick audit,
then drill into the relevant section for implementation detail.

## Source Hierarchy
1. Official case studies (Awwwards, Codrops, Medium by studios)
2. Three.js / R3F / GSAP official documentation
3. WebKit and Chrome bug trackers
4. Primary author blogs (Bruno Simon, Don McCurdy, Evil Martians)
